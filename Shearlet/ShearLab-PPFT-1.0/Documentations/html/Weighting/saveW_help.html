SAVEW

find weight constant and save it as files

Contents

  • See also
  • Copyright

Description

SAVEW(N,R,BASISCHOICE) compute the weight constants in the basis functions and save them as files. N is the image size, R is the oversampling rate, basisChoice is the parameter 'Choice' in BASISFUNCTION.

Examples

   savew(128,2,5);
