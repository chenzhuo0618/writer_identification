Description

 Y = ADJFRFT(X,ALPHA) compute the adjoint fractional Fourier transform
 of X with fraction alpha. This equivalent to Y = FRFT(X,-ALPHA).

Examples

        x = randn(5,3);
        alpha = [1/3,1/2,1/4];
        y = AdjfrFT(x, alpha);
