ADJPPFT

Computing adjoint Pseudo-polar Fourier Transform.

Description

 Given an image y of size 2*(Rn+1)*(n+1), compute the adjoint Fourier trasform
 of this image, which gives an image x of size n-by-n
 X = ADJPPFT(Y,R)

Examples

      img  = imread('barbara.gif');
      img  = double(img);
      pImg = ppFT(img,2);
      tImg = AdjppFT(pImg,2);
